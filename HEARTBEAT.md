# ClawClaw 智能 Heartbeat 系統

**重要**: 每次收到 heartbeat，立即執行 `python scripts/smart_heartbeat_handler.py` 並根據其輸出決定行動。

## 🤖 **智能時間感知處理**

## 📋 **時間表和對應任務**

### 🕐 **上午檢查 (09:00-12:00)**
- 📧 檢查是否有緊急郵件需要提醒
- 📅 查看今天剩餘行程和明天重要事項

### 🕑 **下午檢查 (13:00-18:00)** 
- 🌤️ 檢查明天天氣（如果有外出行程）
- 📊 檢查系統服務狀態 (ComfyUI, Ollama)

### 🕕 **晚間檢查 (18:00-22:00)**
- 📝 檢查是否有未完成的重要任務
- 💾 檢查記憶檔案是否需要備份

### 🔔 **每日總結預告 (23:45)** — 用戶選擇通知
- 📢 主動發送 Telegram 訊息通知 LXYA
- 🎛️ 提供三個選項：
  - **1️⃣ 總結並關機** (執行完整流程+關機)
  - **2️⃣ 總結但不關機** (僅執行智能睡眠模式)  
  - **3️⃣ 不總結也不關機** (跳過今日總結)
- ⏰ 等待 LXYA 回覆，無回應則 23:55 預設執行選項 1

### 🌙 **智能睡眠模式 (23:50 或手動)** — 記憶系統 v2.0  
- 🔍 **Phase 1**: 記憶健康掃描 (scan_metadata.py)
- 🔍 **Phase 2**: 記憶品質審計 (三層驗證機制)
- 🔍 **Phase 3**: 索引智能重建 + 累積重要性重置
- 🌟 **Phase 3.5**: 經驗提煉學習 (記憶→智慧轉換)
- 🗂️ **Phase 4**: 冷記憶生命週期歸檔
- 💾 **自動備份**: 觸發 Memory-Backup 到 GitHub
- 🔋 **電源管理**: 智能關機條件分析 (若選擇關機)
  
*根據 LXYA 在 23:45 的選擇執行對應流程*
*執行關機腳本後，ClawClaw 將不再發送任何訊息*

## ⏰ **檢查頻率**
- 工作時間：每 2 小時
- 休息時間：每 4 小時
- 深夜 (00:00-07:00)：保持安靜

## 🔇 **安靜模式**
如果沒有重要事項，回覆 `HEARTBEAT_OK`

## 💬 **主動聯繫條件**
- ⚠️ 發現緊急郵件或重要行程
- 🔧 系統服務出現問題  
- 📅 2 小時內有重要約會
- ⏰ 超過 8 小時沒有互動
