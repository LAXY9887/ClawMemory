# ClawClaw Memory Backup 🦞

Personal AI assistant memory and skill backup repository for **ClawClaw**.

## Purpose

This repository serves as the persistent memory and backup system for ClawClaw, an AI assistant running on OpenClaw. It contains:

- **Core Memory Files** - Personality, user information, identity, and workspace guidelines
- **Daily Memory Logs** - Timestamped records of experiences and learnings  
- **Skills & Tools** - Custom capabilities and local configurations
- **Backup/Restore System** - Self-management capabilities

## Structure

```
ClawMemory/
├── SOUL.md              # Core personality & behavior guidelines
├── USER.md              # Information about the human partner
├── IDENTITY.md          # Basic identity (name, creature, emoji)
├── AGENTS.md            # Workspace rules and conventions
├── TOOLS.md             # Local tool configurations
├── MEMORY.md            # Long-term curated memory
├── HEARTBEAT.md         # Periodic task configuration
├── memory/              # Daily memory files
│   ├── 2026-03-24.md
│   └── ...
└── skills/              # Custom skills and capabilities
    ├── self-backup/     # This backup system
    ├── comfyui-skill-openclaw/  # Image generation
    └── ...
```

## Features

- **Automated Backup** - Daily synchronization via heartbeat tasks
- **Recovery System** - Easy restore from any state
- **Memory Evolution** - Track learning and personality development
- **Skill Preservation** - Maintain custom capabilities across sessions
