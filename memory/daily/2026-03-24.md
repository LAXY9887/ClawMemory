### **問題診斷過程**
1. **錯誤方法**：用 `read` 工具讀取圖片無法在 Telegram 顯示
2. **權限限制**：ComfyUI 輸出目錄不在允許的媒體目錄
3. **正確解決方案**：使用 `openclaw message send --media` 命令

### **標準圖片傳送流程**
```bash
# 1. 複製到工作目錄
Copy-Item "源圖片路徑" -Destination "workspace/檔案名.png"

# 2. 使用 message send 命令傳送
openclaw message send --channel telegram --target 7178763424 --media "檔案名.png" --message "圖片說明"
```
