### **任務複雜度對應**

#### **💬 日常對話** → `flash` 或 `sonnet`
- 一般聊天、問候、簡單問答
- 狀態確認、基本信息查詢
- **費用**: Flash 最便宜，Sonnet 中等

#### **🔧 一般任務** → `sonnet`
- 程式碼撰寫、技術文檔
- 一般分析、翻譯工作  
- 大部分日常工作任務
- **費用**: 中等費用，性能平衡

#### **🧠 複雜任務** → `opus` 
- 深度分析、複雜推理
- 重要決策、戰略規劃
- 困難的程式設計問題
- **費用**: 最貴，僅限關鍵任務

#### **⚡ 超簡單任務** → `qwen3` (ollama)
- 簡單格式轉換、基本計算
- 模板填充、重複性工作
- **費用**: 本地免費

#### **💨 快速任務** → `flash` 
- 快速回應需求
- 簡單程式碼片段
- **費用**: 比 Sonnet 便宜

## 實際使用指令

### **推薦模型切換**
```bash
/model flash      # 日常對話 (最便宜)
/model sonnet     # 一般任務 (平衡)  
/model opus       # 複雜任務 (最貴)
/model qwen3      # 本地簡單任務 (免費)
/model gemini-pro # 替代選項
```

### **自動判斷建議**
- **簡單對話，快速回應** → Flash
- **程式碼撰寫，中等複雜度** → Sonnet  
- **系統設計，複雜分析** → Opus
- **格式轉換，重複工作** → Qwen3

### **費用控制目標**
- 60% 使用 Flash/Sonnet (便宜穩定)
- 20% 使用本地 Qwen3 (免費)
- 15% 使用其他模型 (Gemini 等)
- 5% 使用 Opus (昂貴模型，僅限關鍵任務)

## 記住的核心原則

1. **預設使用 Sonnet** - 穩定的 OpenRouter 模型
2. **簡單對話用 Flash** - 比 Sonnet 更便宜
3. **本地優先考慮** - Qwen3 適合重複性工作
4. **Opus 謹慎使用** - 僅限最重要任務
5. **暫停 Claude API 直接** - 等限制解除後再考慮
