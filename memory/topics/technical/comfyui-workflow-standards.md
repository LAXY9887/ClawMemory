## LXYA 指定的工作流程標準

### **核心要求**
> "以後 ComfyUI 的流程都要加上本地模型優化提示詞，不管是新的流程還是用現有的流程"

### **標準工作流程**

#### 1. 提示詞智能優化 (必備步驟)
```python
# 使用 Ollama qwen3:8b 模型優化
python scripts/prompt_optimizer.py --input "原始提示詞"
```

#### 2. 高品質圖像生成
```python  
# 使用優化後提示詞 + 高解析度工作流程
python scripts/hires_ollama_generator.py -i "優化後的提示詞"
```

#### 3. 進度監控 (新標準)
```python
# 同時啟動進度監控
python scripts/progress_monitor.py --continuous
```

### **技術組合標準**

#### **核心技術棧**
- **本地優化**: Ollama qwen3:8b (提示詞增強)
- **圖像生成**: ComfyUI + rinIllusion 模型
- **解析度**: 840x1200 (高品質標準)
- **監控**: 實時進度可視化

#### **品質保證**
- 每個提示詞都要經過 Ollama 智能優化
- 所有新工作流程都要整合本地模型
- 現有工作流程要改造升級加入優化步驟
- 提供進度可視化避免「停止誤解」
