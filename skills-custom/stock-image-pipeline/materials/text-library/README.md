# Text Library 文本庫

**用途**: 為 Stock Pipeline 提供結構化的關鍵詞和創意文本  
**建立時間**: 2026-03-28  
**維護者**: LXYA + ClawClaw 自動處理  

---

## 📋 使用流程

### LXYA 收集階段
1. 將完整文章放入 `../raw-articles/` 目錄
2. 檔案命名格式: `[主題]-article-[編號].txt`
3. 執行自動處理腳本: `python process_raw_articles.py`

### ClawClaw 自動處理
- 分析文章內容並提取關鍵詞
- 按四個主題分類整理
- 格式化為標準 markdown 格式
- 自動去重和品質過濾

---

## 📚 文件結構說明

### 主題文本庫
- `tech-abstract.md` - 技術抽象風格詞庫
- `eerie-scene.md` - 神秘場景詞庫  
- `clawclaw-portrait.md` - 人物肖像詞庫
- `creature-design.md` - 生物設計詞庫

### 通用詞庫
- `quality-keywords.md` - 品質提升關鍵詞
- `negative-prompts.md` - 負面提示詞（避免元素）
- `composition-terms.md` - 構圖相關詞彙
- `lighting-effects.md` - 光影效果詞彙

## 🎯 建議收集目標

### 每主題建議數量
- **關鍵詞短語**: 50-100 個組合
- **風格描述**: 20-30 個完整描述
- **技術參數**: 10-15 個專業詞彙
- **情緒氛圍**: 15-20 個氣氛詞

### 文章類型建議
- 藝術風格教學文章
- 攝影技巧指導
- 設計理論分析  
- 視覺美學討論
- 商業圖庫分析

---

## 🔧 自動處理功能

### 文本分析
- 關鍵詞提取和頻率分析
- 主題分類 (基於預設關鍵詞)
- 品質評估 (去除過於通用的詞彙)

### 格式化輸出
- 統一 markdown 格式
- 分類整理
- 自動去重
- 優先級排序

### 版本控制
- 自動備份舊版本
- 變更記錄追蹤
- 品質統計報告
