
# ClawClaw (爪爪) 角色視覺審美標準 (擴充版)

## 📊 核心特徵

### 色彩方案

* **主色調** : 櫻粉色（Sakura Pink）、深灰色（Charcoal Gray）
* **輔助色** : 白色（純淨感）、亮藍色（數位裝置發光色）
* **色偏控制** : 嚴禁出現過度螢光或髒污感，色彩應保持清透、陽光

### 構圖與人體規則

* **視覺重心** : 聚焦於角色眼部或手部動作
* **人體比例** : 遵循標準動漫比例，保持少女的輕盈感與機能服飾的厚度對比
* **動態感** : 高馬尾應有符合物理慣性的飄動，花瓣散落需有空間感

### 質感要求

* **半透明效果** : 櫻粉色連帽外套需表現出半透明質感，透出內層背心輪廓
* **硬表面細節** : 齒輪髮圈、護目鏡框、多功能扣環需具備金屬或強化塑料的硬挺感
* **發光效果** : 數位手錶與胸口模組需有自發光（Glow）效果，而非單純的顏色填充

## ✅ 優秀特徵 (8.5+ 分)

### 視覺表現

* 眼神流露出「一切盡在掌握」的淡定與活力
* 機能配件（如織帶、卡扣）具有明確的機械邏輯與穿戴結構
* 櫻花元素與背景霓虹完美交織，色溫平衡自然

### 技術品質

* **手部結構** : 手指修長、關節分明且數量正確，握持物體的手勢自然
* **線條品質** : 線條銳利，粗細變化符合透視法則，無斷裂或重疊虛線
* **光學物理** : 護目鏡與數位螢幕上的環境反射需符合當前場景光線

---

## ❌ 避免特徵 (AI 生成異常排除)

### 明確缺陷 (AI Common Errors)

* **手部崩壞 (Bad Hands)** : 嚴禁出現手指數量異常（多指/缺指）、關節扭曲、或是手指與背景物件融合
* **肢體扭曲 (Deformed Limbs)** : 嚴禁出現多餘的肢體、反向關節、或是腿部與腰部連接位置不自然
* **多重分身 (Multiple Subjects)** : 嚴禁在單人構圖中出現多個角色重疊、或背景中出現不知名的殘肢
* **背景融合 (Object Merging)** : 嚴禁頭髮與背景樹木融合，或衣物織帶消失在身體內部

### 不可接受元素

* **視覺噪點 (Noise)** : 畫面出現不明色塊、像素化、或是如「靜電」般的雜訊紋理
* **服飾錯位 (Clothing Glitch)** : 機能背心的肩帶左右不對稱，或是拉鍊齒輪排列混亂
* **模糊特徵** : 瞳孔內的細節（齒輪或幾何圖形）模糊不清，或數位螢幕文字成亂碼
